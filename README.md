# Description
Use the deep learning recursive neural network keras RNN-LSTM to search for stocks that rise from the next day on multiple stocks.

keras 2.2.5 LSTM<br>
Ubuntu 18.04.3 LTS<br>

Numpy 1.17.3<br>
Pandas 0.25.2<br>
